# Market Scout
A simple Python project for capturing and validating market insights.

## Usage
1. Run `python -m market_scout` to start the program.
2. Follow the prompts to provide feedback on a report.
3. The program will display the total number of validated insights.
4. The data will be exported to a file named `export.json` in the `data` directory.

## Testing
Run `python -m pytest` to run the tests.

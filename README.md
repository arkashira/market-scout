# Market Scout AI-Powered Competitor Landscape Generation
This project generates a ranked list of top 10 competitors with key metrics for a given market keyword.
## Usage
1. Run `python -m market_scout` to start the program.
2. Enter a market keyword when prompted.
3. The program will output a JSON list of competitors with their name, description, funding, employee count, and product focus.
## Testing
Run `python -m pytest` to execute the tests.

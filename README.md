# Insight
A simple insight management system.

## Usage
1. Create an instance of the `InsightStore` class, passing in a directory to store the insights.
2. Create an `Insight` object and save it to the store using the `save_insight` method.
3. Load all insights from the store using the `load_insights` method.
4. Delete an insight from the store using the `delete_insight` method.
5. Duplicate an insight in the store using the `duplicate_insight` method.

# Market Scout

A Python project that provides a competitor landscape analysis.

## Usage

1. Create an instance of the `MarketScout` class, passing in a dictionary of API keys.
2. Call the `get_competitor_data` method to retrieve competitor data for a given company ID.
3. Call the `get_competitor_landscape` method to retrieve a list of competitor data for a list of company IDs.

## Testing

Run the tests using `pytest`.
